# Minimal Spool holder for mostly drybox

## GOALS
- Labor/material efficient spool holder.
- "Drybox"

## DESIGN:

- PTFE Hole 4.3mm
- Ridge/Rib should be > 2mm and > 45deg
- Calculated spool and support distances in Fusion 360.
